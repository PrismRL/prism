stackLimit: number¶
stack(self: Item, actor: Actor)¶

Stacks an actor into this item. Updates the stackCount of both this item and the input item.

split(self: Item, count: integer) → split: Actor¶

Splits the stack, constructing a new actor with the correct count and returning it. Returns it’s owner if count is 1 and the item is not stackable.

getWeight(self: Item) → weight: number¶

Gets the total weight of this item taking into account it’s stackCount.

getVolume(self: Item) → volume: number¶

Gets the total volume of this item taking into account it’s stackCount.
