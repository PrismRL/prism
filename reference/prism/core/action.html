class Action : Object¶

An ‘Action’ is a command that affects a discrete change in the game state. An Action consists of an owner, a name, a list of targets, and a list of target objects. See Target for more.

owner: Actor¶

The actor taking the action.

protected targets: Target[]¶

(static) A list of targets to apply the action to.

protected targetObjects: Object[]¶

The objects that correspond to the targets.

protected requiredComponents: Component[]¶

(static) Components required for an actor to take this action.

__new(self: Action, owner: Actor, ...: Object)¶

Constructor for the Action class.

Parameters:

  • owner (Actor) – The actor that is performing the action.

  • ... (Object) – An optional list of target actors. Not all actions require targets.

protected canPerform(self: Action, level: Level, ...: unknown) → canPerform: boolean, error?: string¶

Checks if the action is valid and can be executed in the given level. Override this.

Parameters:

level (Level) – The level the action would be performed in.

Returns:

  • canPerform (boolean) – True if the action could be performed, false otherwise.

  • error? (string) – An optional error message, if the action cannot be performed.

hasRequisiteComponents(self: Action, actor: Actor) → hasRequisiteComponents: boolean, missingComponent?: string¶

Checks whether or not the actor has the required components to perform this action.

Parameters:

actor (Actor) – The actor to check.

Returns:

  • hasRequisiteComponents (boolean) – True if the actor has the required components, false otherwise.

  • missingComponent? (string) – The name of the first missing component, should any be missing.

protected perform(self: Action, level: Level, ...: unknown)¶

Performs the action on the level. Override this.

Parameters:

level (Level) – The level to perform the action in.

n (number) – The index of the target actor to retrieve.

target (any) – The target actor at the specified index.

Returns the number of targets associated with this action.

numTargets (number) – The number of targets associated with this action.

getTargetObject(self: Action, index: number) → targetObject: Target | nil¶

Returns the target object at the specified index.

Parameters:

index (number) – The index of the target object to retrieve.

hasTarget(self: Action, actor: Actor) → boolean¶

Determines if the specified actor is a target of this action.

Parameters:

actor (Actor) – The actor to check if they are a target of this action.

Returns:

_1 (boolean) – True if the specified actor is a target of this action, false otherwise.

validateTarget(self: Action, n: number, level: any, owner: Actor, toValidate: any, previousTargets?: any[]) → boolean¶

Validates the specified target for this action.

Parameters:

  • n (number) – The index of the target object to validate.

  • owner (Actor) – The actor that is performing the action.

  • toValidate (any) – The target object to validate.

  • previousTargets? (any[]) – The previously selected targets.

Returns:

_1 (boolean) – True if the specified target actor is valid for this action, false otherwise.
