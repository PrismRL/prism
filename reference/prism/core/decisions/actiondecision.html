global class ActionDecision: Decision¶

Represents an action to be taken, generally by the player. PlayerController yields one to decide what action to perform.

action: Action | nil¶

The action to perform.

__new(self: ActionDecision, actor: Actor)¶
setAction(self: ActionDecision, action: Action)¶

Sets an action. It must be valid, i.e. Level:canPerform() returns true.

Parameters:

action (Action) – A valid action.

trySetAction(self: ActionDecision, action: Action, level: Level): (set: boolean, err?: string)¶

Sets an action if it can be performed and there is not an action set already.

Parameters:

  • action (Action) – An action to try setting.

  • level (Level) – The level.

Returns:

  • set (boolean) – True if the action was set, false otherwise.

  • err? (string) – An error message if setting the action failed.
