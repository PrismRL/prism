class Entity : Object¶

The superclass of entitys and cells, holding their components.

components: Component[]¶

A table containing all of the entity’s component instances. Generated at runtime.

componentCache: table¶

This is a cache for component queries, reducing most queries to a hashmap lookup.

__new(self: Entity)¶

Constructor for an entity. Initializes and copies the entity’s fields from its prototype.

position: Vector2¶
initialize(self: Entity) → Component[]¶

Creates the components for the entity. Override this.

addComponent(self: Entity, component: Component)¶

Adds a component to the entity. This function will check if the component’s prerequisites are met and will throw an error if they are not, or if the entity already has the component.

Parameters:

component (Component) – The component to add to the entity.

removeComponent(self: Entity, component: Component) → unknown¶

Removes a component from the entity. This function will throw an error if the component is not present on the entity.

Parameters:

component (Component) – The component to remove from the entity.

hasComponent(self: Entity, prototype: any) → hasComponent: boolean¶

Returns a bool indicating whether the entity has a component of the given type.

Parameters:

prototype (any) – The prototype of the component to check for.

getComponent(self: Entity, prototype: <T>) → <T>?¶

Searches for a component that inherits from the supplied prototype.

Parameters:

prototype (<T>) – The type of the component to return.

expectComponent(self: Entity, prototype: <T>) → <T>¶

Expects a component, returning it or erroring on nil.

Parameters:

prototype (<T>) – The type of the component to return.
