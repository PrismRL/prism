getBounds(self: LevelBuilder): Vector2, Vector2¶

Returns the location of the upper left and lower right points that exist in the level, in the level’s local coordinate system. Notably, this is based on cell locations. So if an actor is located somewhere without a cell, this method will not consider it.

normalize(self: LevelBuilder)¶

Shifts all cells and actors so the minimum bound is (0, 0).
