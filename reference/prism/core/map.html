class Map : Grid¶

A map manager class that extends the Grid class to handle map-specific functionalities.

opacityCache: BooleanBuffer¶

Caches the opaciy of the cell + actors in each tile for faster fov calculation.

passableCache: BitmaskBuffer¶

A class representing a 2D bitmask buffer using 16-bit integers.

__new(self: Map, w: number, h: number, initialValue: Cell)¶

The constructor for the ‘Map’ class. Initializes the map with the specified dimensions and initial value, and sets up the opacity caches.

Parameters:

  • w (number) – The width of the map.

  • h (number) – The height of the map.

  • initialValue (Cell) – The initial value to fill the map with.

updateCaches(self: Map, x: number, y: number)¶

Updates the opacity cache at the specified coordinates.

Parameters:

  • x (number) – The x-coordinate.

  • y (number) – The y-coordinate.
