class MapBuilder : SparseGrid, IQueryable, SpectrumAttachable¶

A map builder class that extends the SparseGrid class to handle map-specific functionalities.

actors: ActorStorage¶

A list of actors present in the map.

initialValue: fun(): Cell¶

The initial value to fill the map with.

__new(self: MapBuilder, initialValue: fun(): Cell)¶

The constructor for the ‘MapBuilder’ class. Initializes the map with an empty data table and actors list.

Parameters:

initialValue (fun(): Cell) – A cell factory to define the default value of the map.

drawRectangle(self: MapBuilder, x1: number, y1: number, x2: number, y2: number, cellFactory: fun(): Cell)¶

Draws a rectangle on the map.

Parameters:

  • x1 (number) – The x-coordinate of the top-left corner.

  • y1 (number) – The y-coordinate of the top-left corner.

  • x2 (number) – The x-coordinate of the bottom-right corner.

  • y2 (number) – The y-coordinate of the bottom-right corner.

  • cellFactory (fun(): Cell) – The cell factory to fill the rectangle with.

drawEllipse(self: MapBuilder, cx: number, cy: number, rx: number, ry: number, cellFactory: fun(): Cell)¶

Draws an ellipse on the map.

Parameters:

  • cx (number) – The x-coordinate of the center.

  • cy (number) – The y-coordinate of the center.

  • rx (number) – The radius along the x-axis.

  • ry (number) – The radius along the y-axis.

  • cellFactory (fun(): Cell) – The cell factory to fill the ellipse with.

drawLine(self: MapBuilder, x1: number, y1: number, x2: number, y2: number, cellFactory: fun(): Cell)¶

Draws a line on the map using Bresenham’s line algorithm.

Parameters:

  • x1 (number) – The x-coordinate of the starting point.

  • y1 (number) – The y-coordinate of the starting point.

  • x2 (number) – The x-coordinate of the ending point.

  • y2 (number) – The y-coordinate of the ending point.

  • cellFactory (fun(): Cell) – The cell factory to draw the line with.

drawPolygon(self: MapBuilder, cellFactory: fun(): Cell, ...: integer)¶

Draws a sequence of lines between given points.

Parameters:

  • cellFactory (fun(): Cell) – The cell factory to draw the lines with.

  • ... (integer) – Pairs of (x, y) coordinates given as a sequence of numbers.

addPadding(self: MapBuilder, width: number, cellFactory: fun(): Cell)¶

Adds padding around the map with a specified width and cell value.

Parameters:

  • width (number) – The width of the padding to add.

  • cellFactory (fun(): Cell) – The cell factory to use for padding.

blit(self: MapBuilder, source: MapBuilder, destX: number, destY: number, maskFn: fun(x: integer, y: integer, source: Cell, dest: Cell) | nil)¶

Blits the source MapBuilder onto this MapBuilder at the specified coordinates.

Parameters:

  • source (MapBuilder) – The source MapBuilder to copy from.

  • destX (number) – The x-coordinate of the top-left corner in the destination MapBuilder.

  • destY (number) – The y-coordinate of the top-left corner in the destination MapBuilder.

  • maskFn (fun(x: integer, y: integer, source: Cell, dest: Cell) | nil) – A callback function for masking. Should return true if the cell should be copied, false otherwise.

build(self: MapBuilder) → Map, table¶

Builds the map and returns the map and list of actors. Converts the sparse grid to a contiguous grid.

Returns:

  • _1 (Map) – actors map and the list of actors.

  • _2 (table) – actors map and the list of actors.
