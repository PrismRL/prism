class SystemManager: Object¶
systems: System[]¶
__new(self: SystemManager, owner: Level)¶
owner: Level¶

Represents a single game level, managing the map, actors, systems, scheduling, and cached data for FOV and pathfinding. Also handles the turn-based game loop via run and step.

onYield(self: SystemManager, level: Level, event: Message)¶

Calls the on yield method for each system right before the level hands a Decision back to the interface. Used by the Sight system to ensure that the player’s fov is always updated when we yield even if it’s not their turn.

Parameters:

  • level (Level) – The level to call onYield for.

  • event (Message) – The event data that caused the yield.
