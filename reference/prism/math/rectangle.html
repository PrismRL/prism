class Rectangle: Object¶

A Rectangle represents a 2D rectangular area with a position, width, and height.

position: Vector2¶

The top-left corner position of the rectangle.

width: number¶

The width of the rectangle.

height: number¶

The height of the rectangle.

    self: Rectangle,

    y: number,
    width: number,
    height: number

Constructor for Rectangle.

Parameters:

  • x (number) – The x-coordinate of the top-left corner, or a Vector2 for the position.

  • y (number) – The y-coordinate of the top-left corner (if xOrPosition is a number).

  • width (number) – The width of the rectangle.

  • height (number) – The height of the rectangle.

contains(self: Rectangle, point: Vector2): (contains: boolean)¶

Checks if a given point is inside the rectangle. The point is considered inside if it’s within or on the boundaries.

Parameters:

point (Vector2) – The point to check.

Returns:

contains (boolean) – True if the point is inside, false otherwise.

intersects(self: Rectangle, other: Rectangle): (intersects: boolean)¶

Checks if this rectangle intersects with another rectangle. Returns true if the rectangles overlap.

Parameters:

other (Rectangle) – The other rectangle to check against.

Returns:

intersects (boolean) – True if the rectangles intersect, false otherwise.

union(self: Rectangle, other: Rectangle): (union: Rectangle)¶

Creates a new rectangle that is the union of this rectangle and another. The union is the smallest rectangle that contains both rectangles.

Parameters:

other (Rectangle) – The other rectangle to unite with.

Returns:

union (Rectangle) – A new Rectangle instance representing the union.

toCorners(self: Rectangle): (corners: table<Vector2>)¶

Returns the four corner points of the rectangle. The order is typically top-left, top-right, bottom-right, bottom-left.

Returns:

corners (table<Vector2>) – A table containing the four Vector2 corner points.
