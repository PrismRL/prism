class Queue : Object¶

A basic FIFO (First In, First Out) queue implementation.

Initializes a new Queue instance.

push(self: Queue, value: any)¶

Adds an element to the end of the queue.

Parameters:

value (any) – The value to be added to the queue.

pop(self: Queue) → any¶

Removes and returns the element from the start of the queue.

Returns:

_1 (any) – The value at the start of the queue.

empty(self: Queue) → boolean¶

Checks if the queue is empty.

Returns:

_1 (boolean) – True if the queue is empty, false otherwise.

peek(self: Queue) → any¶

Returns the element at the start of the queue without removing it.

Returns:

_1 (any) – The value at the start of the queue.

clear(self: Queue)¶

Removes all elements from the queue.

Checks if the queue contains a specific value.

Parameters:

value (any) – The value to check for.

size(self: Queue) → number¶

Returns:

remove(self: Queue, value: any) → boolean¶
